# Career Copilot AI

Career Copilot AI is a full-stack AI-powered platform designed to help job seekers improve their employability through resume analysis, ATS scoring, skill-gap detection, job matching, mock interviews, and personalized learning roadmaps.

## Features

- Resume Upload & Analysis
- ATS Score Evaluation
- Skill Gap Detection
- AI Job Matching
- Mock Interview Assistant
- Personalized Learning Roadmaps
- Application Tracking Dashboard
